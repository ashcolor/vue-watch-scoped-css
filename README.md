# Vue watch scoped css

1. Npm install

  ```
  npm install
  ```

2. Build with watch option.

  ```
  # vite build --watch
  npm run watch
  ```

3. Preview and Open browser.

  ```
  # another terminal
  npm run preview
  ```

4. Change `src\components\HelloWorld.vue` anything, and rebuld automatically.

5. Reload browser, a {color: #42b983;} is not worked.
